Deploy Automatizado com GitHub Actions

Fabrício Moreira – 2011075131
Paula Lopes - 2009076370

Esta aplicação exemplifica a implementação de uma Integração e Entrega Contínuas (CI/CD) utilizando GitHub Actions, com construção e publicação automática de uma imagem Docker no GitHub Container Registry, seguida da sua execução numa máquina virtual.

Como funciona o pipeline:

  • 1. Commit para o repositório GitHub
  • 2. GitHub Actions compila o código e cria a imagem Docker
  • 3. A imagem é publicada no GitHub Container Registry (GHCR)
  • 4. A imagem é instalada e lançada numa máquina virtual do grupo com o seguinte comando: 
    docker pull ghcr.io/paulalopesdeveloper/github-actions-hello-test:latest

docker run -d -p 80:80 ghcr.io/paulalopesdeveloper/github-actions-hello-test:latest
